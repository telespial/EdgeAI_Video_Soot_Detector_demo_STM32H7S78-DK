# Project MRD Assets (ST Scope)

This project keeps local STMicroelectronics documents required for the STM32H7S78-DK soot detector demo.

## Structure
- `stmicroelectronics/platforms/STM32H7S78-DK/pdf/`
- `stmicroelectronics/mcus/STM32H7RS/pdf/`
- `stmicroelectronics/mcus/STM32H7S3A8/pdf/`

## Rule
Only keep ST documents relevant to this project in this local `mrd/` folder.
Use `mrd_library/` for broader cross-project catalogs.
